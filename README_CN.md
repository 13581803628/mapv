## 简介 [English](https://github.com/huiyan-fe/mapv/blob/v2/README.md)
Mapv 是一款地理信息可视化开源库，可以用来展示大量地理信息数据，点、线、面的数据，每种数据也有不同的展示类型，如直接打点、热力图、网格、聚合等方式展示数据。
